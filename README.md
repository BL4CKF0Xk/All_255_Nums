 # Usage
 you have to install "gcc" for compile the c file or you can file i compiled

 ```bash
    gcc -o Print_All_255_numbers Print_All_255_numbers.c
 ```
 And after that you can run the program!
 ```bash
    ./Print_All_255_numbers
 ```
